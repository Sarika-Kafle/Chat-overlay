📖 How to Add to OBS / Streamlabs
1
Host your files (see hosting section below) and get your overlay.html URL
2
In OBS: Sources → + → Browser Source
In Streamlabs: Sources → + → Browser Source
3
Paste your generated URL into the URL field
4
Set Width: 400, Height: 800 (or full canvas). Enable "Custom CSS" → add body { background: transparent !important; } if needed
5
Click OK — chat appears on your stream! 🎉 Drag to position it anywhere.
